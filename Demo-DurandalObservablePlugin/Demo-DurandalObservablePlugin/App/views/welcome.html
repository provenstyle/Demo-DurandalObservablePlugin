Durandal Observable Plugin

The observable plugin makes plain javascript objects observable!

Plugs in to the page life cycle

Transforms all viewModels during the binding event. The binding event occures right before the view and viewModel are bound.

Can be used directly in your code

convertObject
makes and entire javascript object observable
convertProperty
allows you to make a specific property on an object observable
allows you to add a new observable property to an object
defineProperty
allows you to add computed functions to your observable

Contiues working after the objects have been transformed

  • Any property being set on the viewModel will be made observable
  • Any item pushed to an array on the view will be made observable

Transforms promises into observables

  • If the returned data from the promise is a single property it becomes an observable
  • If the returned data from the promise is an array it becomes an observable array

Makes these types observable

  • Strings
  • Arrays

Ignores these types

  • Functions
For general documenation please visit the observable plugin documentation on the official Durandal site and if you can't find answers to your questions there, join the google group.
